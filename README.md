# PlaywrightLearningBatch

This repository contains JavaScript learning examples and notes organized by chapter. It is designed for learning JavaScript syntax, identifier rules, comments, and related concepts.

## Repository Structure

- `Chapter_01_Basics/`
  - JavaScript basics, setup verification, and introductory examples.
- `Chapter_02_JavaScript_Concepts/`
  - JavaScript concepts, identifier rules, and naming conventions.
- `Chapter_03_Identifier_Literals/`
  - Comments, literal examples, and identifier usage.
- `Chapter_04_JavascriptsConcepts/`
  - Variable declarations, function examples, hoisting, scope, and Temporal Dead Zone (TDZ).
- `Chapter_05_literals/`
  - Literals, null vs undefined, number types, template literals, and string comparisons.
- `Chapter_06_double_triple_equal/`
  - Operators, comparisons, loose equality vs strict equality, logical operators, and ternary operations.
- `Chapter_07_IfElse/`
  - If-else statements, conditional logic, real-world examples, and class exercises.
- `Chapter_08_SwitchStaement/`
  - Switch statements, break statements, default cases, and switch grouping patterns.
- `Chapter_09_Userinput/`
  - User input handling, node readline module, and synchronous prompts.
- `Chapter_10_loops/`
  - For loops, while loops, do-while loops, for-of, for-in, and array iteration methods.
- `Chapter_11_Array/`
  - Array creation, access, manipulation, searching, iteration, and array transformation methods.
- `Chapter_12_Functions/`
  - Function types, parameters, return values, function expressions, arrow functions, closures, and interview questions.

## Notes

This repository is intended for learning and practicing JavaScript fundamentals, covering:
- Identifier conventions and comments
- Variable declarations (var, let, const) and scope
- Temporal Dead Zone (TDZ) with let and const
- Literals: null, undefined, numbers, strings, and template literals
- All number types: integers, floats, hexadecimal, octal, binary, BigInt, and special values
- Operators: comparison, logical, arithmetic, and ternary
- Control flow: if-else statements and conditional logic
- Switch statements and pattern matching
- User input handling and synchronous operations
- Loops: for, while, do-while, for-of, for-in, and iteration methods
- Arrays: creation, access, manipulation, searching, and transformation
- Functions: declarations, expressions, parameters, return values, arrow functions, and closures
- Loose equality (==) vs strict equality (===)
- Keep the repository up to date by committing and pushing all example code.
