# PlaywrightLearningBatch

This repository contains JavaScript learning examples and notes organized by chapter. It is designed for learning JavaScript syntax, identifier rules, comments, and related concepts.

## Repository Structure

- `Chapter_01_Basics/`
  - JavaScript basics, setup verification, and introductory examples.
- `Chapter_02_JavaScript_Concepts/`
  - JavaScript concepts, identifier rules, and naming conventions.
- `Chapter_03_Identifier_Literals/`
  - Comments, literal examples, and identifier usage.
- `Chapter_04_JavascriptsConcepts/`
  - Variable declarations, function examples, hoisting, and scope.

## Notable Files

- `Chapter_02_JavaScript_Concepts/06_IdentifierRules.js` - Examples of allowed and disallowed JavaScript identifiers.
- `Chapter_02_JavaScript_Concepts/07_IdentifierPart2.js` - Identifier naming styles such as camelCase, snake_case, and PascalCase.
- `Chapter_03_Identifier_Literals/08_Comments.js` - Comment examples and VS Code shortcut reference.
- `Chapter_04_JavascriptsConcepts/19_hoisting_functionexample.js` - Function hoisting and scope examples.

## Usage

Open the files in Visual Studio Code to review the examples and experiment by running them using a JavaScript runtime such as Node.js.
